使用 nohup 使得命令后台运行，同时断网和关闭终端都不会终止任务，适合用来作为任务提交方式。

此外，python 中自带的 subprocess.popen() 方法同样可以起到后台运行命令且关闭终端不退出的作用，所以不必要使用 nohup。
